# Images Folder

This folder contains all the images used in The Shape Up Fitness Studio website.

## Required Images:

1. **hero-bg.jpg** - Hero section background image
2. **equipment.jpg** - Modern gym equipment
3. **workout-area.jpg** - Main workout area
4. **members-training.jpg** - Members training
5. **interior.jpg** - Gym interior
6. **training-session.jpg** - Training session in progress
7. **fitness-equipment.jpg** - Various fitness equipment

## Instructions:

- Add your images in this folder
- Make sure they have the exact filenames listed above
- Recommended image sizes:
  - Hero background: 1920x1080px
  - Gallery images: 800x600px
- Use high-quality images for best results
- Images will automatically appear on the website once added

## Image Guidelines:

- Use bright, well-lit photos
- Show clean, modern gym environment
- Include people working out for authenticity
- Maintain consistent style across all images
